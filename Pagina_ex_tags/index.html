Explicação de Tags HTML Básicas

Tags Principais

O elemento <body> contém todo o conteúdo visível da página web.

O elemento <head> contém metadados sobre a página, como título, meta tags, etc.

Elementos de Estruturação

O elemento <h1> define o título principal da página.

O elemento <hr> cria uma linha horizontal.

O elemento <br> cria uma quebra de linha.

Elementos de Navegação

O elemento <a> cria um link para outra página ou seção.

O elemento <ol> cria uma lista ordenada.

O elemento <ul> cria uma lista não-ordenada.

O elemento <li> representa um item em uma lista.

Elementos de Conteúdo

O elemento <p>define um parágrafo de texto.

O elemento <img> insere uma imagem na página.
